# KCU-Hub

KCU-Hub는 고려사이버대학교 소프트웨어학부 학우들이 공부 기록, 스터디 자료, 프로젝트 결과물을 오래 남기기 위해 만든 비공식 GitHub 허브입니다.

기본 언어는 한국어입니다. GitHub를 잘 몰라도 자료를 읽고, 잘못된 링크를 알려주고, 스터디 기록을 남기는 방식으로 참여하실 수 있습니다.

## 먼저 보실 곳

- [학우용 참여 가이드](https://kcu-hub.github.io/.github/)
- [Markdown 가이드](https://github.com/KCU-Hub/.github/blob/main/docs/contributor-guide.md)
- [운영 모델](https://github.com/KCU-Hub/.github/blob/main/docs/operating-model.md)
- [repo 구조 제안](https://github.com/KCU-Hub/.github/blob/main/docs/repo-structure.md)

## 여기서 할 수 있는 일

| 구분 | GitHub에 남기는 것 | GitHub에 남기지 않는 것 |
|---|---|---|
| 과목 기록 | 공개 가능한 요약, 참고 링크, 복습 노트 | 성적, 출석, 개인 상담 |
| 스터디 | 주차별 계획, 발표 자료, 회고 | 실시간 잡담, 즉답성 질문 |
| 프로젝트 | README, 설치 방법, 이슈, 회고 | 비밀키, 내부 계정, 민감한 운영 정보 |
| 운영 | 문서 수정 제안, 새 repo 제안, 공개 로드맵 | 외부 인물 실명, 이메일, 내부 식별자 |

## 참여 방식

1. 자료를 읽고 도움이 되는 링크를 저장합니다.
2. 틀린 내용이나 깨진 링크를 발견하면 Issue로 알려주십니다.
3. 스터디나 프로젝트를 진행했다면 결과를 README나 Markdown으로 남깁니다.
4. 빠른 대화는 외부 채널에서 하고, 나중에 다시 볼 결론만 GitHub에 정리합니다.

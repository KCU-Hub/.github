# KCU-Hub

KCU-Hub는 GitHub org 중심의 지식 전수, 프로젝트, 스터디 허브입니다.

실시간 대화는 카카오톡, 밴드, 네이버 카페 같은 외부 채널에서 진행하고, GitHub에는 오래 남길 산출물과 협업 기록을 정리합니다. 기존 풀스택 커뮤니티 앱은 메인 커뮤니티가 아니라 개별 프로젝트 중 하나로 다룹니다.

## 무엇을 남기나요

| 구분 | GitHub에 남기는 것 | GitHub에 남기지 않는 것 |
|---|---|---|
| 과목 기록 | 과목별 정리, 참고 자료 링크, 복습 노트 | 개인정보가 포함된 상담, 성적, 출석 정보 |
| 스터디 | 커리큘럼, 주차별 기록, 발표 자료 | 실시간 잡담, 즉답성 질문 |
| 프로젝트 | README, 이슈, PR, 회고, 릴리스 노트 | 비밀키, 내부 계정, 민감한 운영 정보 |
| 운영 | repo 제안, 문서 수정, 공개 로드맵 | 외부 인물 실명, 이메일, 내부 식별자 |

## 참여 방식

1. 고칠 문서나 새 repo 아이디어가 있으면 Issue를 엽니다.
2. 오래 남길 산출물은 README, Markdown 문서, PR로 정리합니다.
3. 빠른 대화가 필요한 내용은 외부 채널에서 먼저 합의한 뒤 GitHub에 결과만 남깁니다.
4. 개인 정보나 민감 정보가 들어간 자료는 공개 repo에 올리지 않습니다.
